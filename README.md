# Python-Clean-Architecture-Sample: DDD, CQS, and Hexagonal Patterns.

A technical showcase of advanced architectural patterns in the Python ecosystem.
- Patterns: Domain-Driven Design (DDD), Command-Query Separation (CQS), and Dependency Inversion.
- Design: Vertical Slice Architecture to minimize cross-feature coupling.
- Stack: FastAPI, SQLModel and PostgreSQL.
- Goal: To demonstrate how enterprise-level patterns (common in C#/.NET) can be elegantly implemented in Python.
